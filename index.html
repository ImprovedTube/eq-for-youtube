Installation & Usage Tutorial

Features

  • Six fully parametric bands with adjustable gain, frequency, and Q factor
  • Real-time frequency visualization
  • Built-in presets for common audio enhancement
  • Create and save your own custom presets
  • Automatically remembers settings for each YouTube channel
  • Dark/light mode theme options

Installation

  1. Download the extension using the button above
  2. Unzip the downloaded file
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" in the top-right corner
  5. Click "Load unpacked" and select the unzipped folder
  6. The EQ for YouTube extension is now installed!
